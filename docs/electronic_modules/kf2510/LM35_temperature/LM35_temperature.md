# LM35温度传感器

## 1 简介

这是一款基于LM35的半导体温度传感器，可以用来对环境温度进行检测。半导体温度传感器的应用简单，有较好的线性度和较高的灵敏度。 LM35半导体温度传感器是美国国家半导体公司生产的线性温度传感器。其测温范围是0℃到100℃，灵敏度为10mV/℃，输出电压与温度成正比。
