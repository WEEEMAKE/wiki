### 1.1 简要概述
ELF UNO Shield是由weeemake开发，兼容Arduino的主控板和拓展板。专为机器人应用而设计。适用于电机和其他模块驱动控制的开发套件。控制器主控采用了ATmega328P芯片。主板上提供了丰富的接口，方便接入各种传感器模块，受益Arduino开源平台，因此受到成千上万的开放源码的支持，降低了学习单片机的门槛。
